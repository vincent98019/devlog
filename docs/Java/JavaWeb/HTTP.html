HTTP

WebжҳҜе…Ёзҗғе№ҝеҹҹзҪ‘пјҢд№ҹз§°дёәдёҮз»ҙзҪ‘(www)пјҢиғҪеӨҹйҖҡиҝҮжөҸи§ҲеҷЁи®ҝй—®зҡ„зҪ‘з«ҷгҖӮ

еңЁж—Ҙеёёзҡ„з”ҹжҙ»дёӯпјҢз»ҸеёёдјҡдҪҝз”ЁжөҸи§ҲеҷЁеҺ»и®ҝй—®зҷҫеәҰгҖҒдә¬дёңзӯүиҝҷдәӣзҪ‘з«ҷпјҢиҝҷдәӣзҪ‘з«ҷз»ҹз§°дёәWebзҪ‘з«ҷгҖӮ

иҪҜд»¶жһ¶жһ„вҖӢ

жЎҢйқўеә”з”ЁвҖӢ

жЎҢйқўеә”з”Ёдҝ—з§°еҚ•еҮ»еә”з”ЁпјҢиҪҜд»¶жүҖжңүж•°жҚ®йғҪдҝқеӯҳеңЁз”өи„‘зҡ„жң¬ең°зЎ¬зӣҳдёӯ

  • дјҳзӮ№пјҡ жҳ“дәҺдҪҝз”ЁпјҢз»“жһ„з®ҖеҚ•
  • зјәзӮ№пјҡ ж•°жҚ®йҡҫд»Ҙе…ұдә«пјҢе®үе…ЁжҖ§е·®пјҢжӣҙж–°дёҚеҸҠж—¶

C/SпјҡClient-ServerжЁЎејҸвҖӢ

Client/Serverз»“жһ„(C/Sз»“жһ„)жҳҜжҢҮе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз»“жһ„пјҢеңЁз”ЁжҲ·жң¬ең°жңүдёҖдёӘе®ўжҲ·з«ҜзЁӢеәҸпјҢеңЁиҝңзЁӢжңүдёҖдёӘжңҚеҠЎеҷЁз«ҜзЁӢеәҸ

  • дјҳзӮ№пјҡ ж•°жҚ®ж–№дҫҝе…ұдә«пјҢе®үе…ЁжҖ§й«ҳ
  • зјәзӮ№пјҡ еҝ…йЎ»е®үиЈ…е®ўжҲ·з«ҜпјҢеҚҮзә§дёҺз»ҙжҠӨеӣ°йҡҫ

B/SпјҡBroswer-ServerжЁЎејҸвҖӢ

Broswer-Server(B/S)жЁЎејҸеҚіжөҸи§ҲеҷЁе’ҢжңҚеҠЎеҷЁжһ¶жһ„жЁЎејҸпјҢеҸӘйңҖиҰҒдёҖдёӘжөҸи§ҲеҷЁпјҢз”ЁжҲ·йҖҡиҝҮдёҚеҗҢзҡ„зҪ‘еқҖ(URL)пјҢе®ўжҲ·и®ҝй—®дёҚеҗҢзҡ„жңҚеҠЎеҷЁз«ҜзЁӢеәҸгҖӮ

  • дјҳзӮ№пјҡејҖеҸ‘з®ҖеҚ•пјҢж— йңҖе®үиЈ…е®ўжҲ·з«ҜпјҢж•°жҚ®жҳ“дәҺе…ұдә«
  • зјәзӮ№пјҡзӣёиҫғдәҺC/SжЁЎејҸпјҢжү§иЎҢйҖҹеәҰдёҺз”ЁжҲ·дҪ“йӘҢзӣёеҜ№иҫғејұпјҢеҜ№зЎ¬д»¶иҰҒжұӮиҝҮй«ҳ

е®ўжҲ·з«ҜеҸӘйңҖиҰҒжөҸи§ҲеҷЁпјҢеә”з”ЁзЁӢеәҸзҡ„йҖ»иҫ‘е’Ңж•°жҚ®йғҪеӯҳеӮЁеңЁжңҚеҠЎеҷЁз«ҜгҖӮжөҸи§ҲеҷЁеҸӘйңҖиҰҒиҜ·жұӮжңҚеҠЎеҷЁпјҢиҺ·еҸ–Webиө„жәҗпјҢжңҚеҠЎеҷЁжҠҠWebиө„жәҗеҸ‘йҖҒз»ҷжөҸи§ҲеҷЁеҚіеҸҜгҖӮ

иҜ·жұӮдёҺе“Қеә”вҖӢ

  • д»ҺжөҸи§ҲеҷЁеҸ‘еҮәйҖҒз»ҷжңҚеҠЎеҷЁзҡ„ж•°жҚ®еҢ…з§°дёәвҖңиҜ·жұӮ(Request)вҖқ
  • д»ҺжңҚеҠЎеҷЁиҝ”еӣһз»ҷжөҸи§ҲеҷЁзҡ„з»“жһңз§°дёәвҖңе“Қеә”(Response)вҖқ

иө„жәҗеҲҶзұ»вҖӢ

йқҷжҖҒиө„жәҗвҖӢ

дҪҝз”ЁйқҷжҖҒзҪ‘йЎөејҖеҸ‘жҠҖжңҜеҸ‘еёғзҡ„иө„жәҗ

  • жүҖжңүз”ЁжҲ·и®ҝй—®пјҢеҫ—еҲ°зҡ„з»“жһңжҳҜдёҖж ·зҡ„гҖӮеҰӮпјҡж–Үжң¬пјҢеӣҫзүҮпјҢйҹійў‘гҖҒи§Ҷйў‘, HTML,CSS,JavaScript
  • еҰӮжһңз”ЁжҲ·иҜ·жұӮзҡ„жҳҜйқҷжҖҒиө„жәҗпјҢйӮЈд№ҲжңҚеҠЎеҷЁдјҡзӣҙжҺҘе°ҶйқҷжҖҒиө„жәҗеҸ‘йҖҒз»ҷжөҸи§ҲеҷЁгҖӮ
  • жөҸи§ҲеҷЁдёӯеҶ…зҪ®дәҶйқҷжҖҒиө„жәҗзҡ„и§Јжһҗеј•ж“ҺпјҢеҸҜд»Ҙеұ•зӨәйқҷжҖҒиө„жәҗ

еҠЁжҖҒиө„жәҗвҖӢ

дҪҝз”ЁеҠЁжҖҒзҪ‘йЎөеҸҠж—¶еҸ‘еёғзҡ„иө„жәҗ

  • жүҖжңүз”ЁжҲ·и®ҝй—®пјҢеҫ—еҲ°зҡ„з»“жһңеҸҜиғҪдёҚдёҖж ·гҖӮеҰӮпјҡjsp/servlet,php,asp...
  • еҰӮжһңз”ЁжҲ·иҜ·жұӮзҡ„жҳҜеҠЁжҖҒиө„жәҗпјҢйӮЈд№ҲжңҚеҠЎеҷЁдјҡжү§иЎҢеҠЁжҖҒиө„жәҗпјҢиҪ¬жҚўдёәйқҷжҖҒиө„жәҗпјҢеҶҚеҸ‘йҖҒз»ҷжөҸи§ҲеҷЁ

HTTPвҖӢ

HyperText Transfer ProtocolпјҢи¶…ж–Үжң¬дј иҫ“еҚҸи®®пјҢи§„е®ҡдәҶжөҸи§ҲеҷЁе’ҢжңҚеҠЎеҷЁд№Ӣй—ҙж•°жҚ®дј иҫ“зҡ„и§„еҲҷгҖӮ

  • ж•°жҚ®дј иҫ“зҡ„и§„еҲҷжҢҮзҡ„жҳҜиҜ·жұӮж•°жҚ®е’Ңе“Қеә”ж•°жҚ®йңҖиҰҒжҢүз…§жҢҮе®ҡзҡ„ж јејҸиҝӣиЎҢдј иҫ“гҖӮ
  • еҰӮжһңжғізҹҘйҒ“е…·дҪ“зҡ„ж јејҸпјҢеҸҜд»Ҙжү“ејҖжөҸи§ҲеҷЁпјҢзӮ№еҮ»F12жү“ејҖејҖеҸ‘иҖ…е·Ҙе…·пјҢзӮ№еҮ»NetworkжқҘжҹҘзңӢжҹҗдёҖж¬ЎиҜ·жұӮзҡ„иҜ·жұӮж•°жҚ®е’Ңе“Қеә”ж•°жҚ®е…·дҪ“зҡ„ж јејҸеҶ…е®№гҖӮ

еҺҶеҸІзүҲжң¬вҖӢ

  • 1.0пјҡжҜҸдёҖж¬ЎиҜ·жұӮе“Қеә”йғҪдјҡе»әз«Ӣж–°зҡ„иҝһжҺҘ
  • 1.1пјҡеӨҚз”ЁиҝһжҺҘ

HTTPеҚҸи®®зү№зӮ№вҖӢ

  • еҹәдәҺTCPеҚҸи®®пјҡ йқўеҗ‘иҝһжҺҘпјҢе®үе…Ё

TCPжҳҜдёҖз§Қйқўеҗ‘иҝһжҺҘзҡ„(е»әз«ӢиҝһжҺҘд№ӢеүҚжҳҜйңҖиҰҒз»ҸиҝҮдёүж¬ЎжҸЎжүӢ)гҖҒеҸҜйқ зҡ„гҖҒеҹәдәҺеӯ—иҠӮжөҒзҡ„дј иҫ“еұӮйҖҡдҝЎеҚҸи®®пјҢеңЁж•°жҚ®дј иҫ“ж–№йқўжӣҙе®үе…ЁгҖӮ

  • еҹәдәҺиҜ·жұӮ-е“Қеә”жЁЎеһӢзҡ„пјҡ дёҖж¬ЎиҜ·жұӮеҜ№еә”дёҖж¬Ўе“Қеә”

иҜ·жұӮе’Ңе“Қеә”жҳҜдёҖдёҖеҜ№еә”е…ізі»

  • HTTPеҚҸи®®жҳҜж— зҠ¶жҖҒеҚҸи®®пјҡ еҜ№дәҺдәӢзү©еӨ„зҗҶжІЎжңүи®°еҝҶиғҪеҠӣгҖӮжҜҸж¬ЎиҜ·жұӮ-е“Қеә”йғҪжҳҜзӢ¬з«Ӣзҡ„

ж— зҠ¶жҖҒжҢҮзҡ„жҳҜе®ўжҲ·з«ҜеҸ‘йҖҒHTTPиҜ·жұӮз»ҷжңҚеҠЎз«Ҝд№ӢеҗҺпјҢжңҚеҠЎз«Ҝж №жҚ®иҜ·жұӮе“Қеә”ж•°жҚ®пјҢе“Қеә”е®ҢеҗҺпјҢдёҚдјҡи®°еҪ•д»»дҪ•дҝЎжҒҜгҖӮиҝҷз§Қзү№жҖ§жңүдјҳзӮ№д№ҹжңүзјәзӮ№пјҢзјәзӮ№пјҡеӨҡж¬ЎиҜ·жұӮй—ҙдёҚиғҪе…ұдә«ж•°жҚ®пјӣдјҳзӮ№пјҡйҖҹеәҰеҝ«гҖӮ

иҜ·жұӮж•°жҚ®ж јејҸвҖӢ

иҜ·жұӮж•°жҚ®жҖ»е…ұеҲҶдёәдёүйғЁеҲҶеҶ…е®№пјҢеҲҶеҲ«жҳҜиҜ·жұӮиЎҢгҖҒиҜ·жұӮеӨҙгҖҒиҜ·жұӮдҪ“

иҜ·жұӮиЎҢвҖӢ

HTTPиҜ·жұӮдёӯзҡ„з¬¬дёҖиЎҢж•°жҚ®пјҢиҜ·жұӮиЎҢеҢ…еҗ«дёүеқ—еҶ…е®№пјҢеҲҶеҲ«жҳҜGET[иҜ·жұӮж–№ејҸ] /[иҜ·жұӮURLи·Ҝеҫ„] HTTP/1.1[HTTPеҚҸи®®еҸҠзүҲжң¬]пјҢиҜ·жұӮж–№ејҸжңүдёғз§ҚпјҢжңҖеёёз”Ёзҡ„жҳҜGETе’ҢPOST

иҜ·жұӮеӨҙвҖӢ

з¬¬дәҢиЎҢејҖе§ӢпјҢж јејҸдёәkey: valueеҪўејҸпјҢиҜ·жұӮеӨҙдёӯдјҡеҢ…еҗ«иӢҘе№ІдёӘеұһжҖ§пјҢеёёи§Ғзҡ„HTTPиҜ·жұӮеӨҙжңүпјҡ

HostпјҡиЎЁзӨәиҜ·жұӮзҡ„дё»жңәеҗҚ
User-AgentпјҡжөҸи§ҲеҷЁзүҲжң¬,дҫӢеҰӮChromeжөҸи§ҲеҷЁзҡ„ж ҮиҜҶзұ»дјјMozilla/5.0 ...Chrome/79пјҢIEжөҸи§ҲеҷЁзҡ„ж ҮиҜҶзұ»дјјMozilla/5.0 (Windows NT ...)like Geckoпјӣ
AcceptпјҡиЎЁзӨәжөҸи§ҲеҷЁиғҪжҺҘж”¶зҡ„иө„жәҗзұ»еһӢпјҢеҰӮtext/*пјҢimage/*жҲ–иҖ…*/*иЎЁзӨәжүҖжңүпјӣ
Accept-LanguageпјҡиЎЁзӨәжөҸи§ҲеҷЁеҒҸеҘҪзҡ„иҜӯиЁҖпјҢжңҚеҠЎеҷЁеҸҜд»ҘжҚ®жӯӨиҝ”еӣһдёҚеҗҢиҜӯиЁҖзҡ„зҪ‘йЎөпјӣ
Accept-EncodingпјҡиЎЁзӨәжөҸи§ҲеҷЁеҸҜд»Ҙж”ҜжҢҒзҡ„еҺӢзј©зұ»еһӢпјҢдҫӢеҰӮgzip, deflateзӯүгҖӮ

иҜ·жұӮдҪ“вҖӢ

POSTиҜ·жұӮзҡ„жңҖеҗҺдёҖйғЁеҲҶпјҢеӯҳеӮЁиҜ·жұӮеҸӮж•°

еҰӮдёҠеӣҫзәўзәҝжЎҶзҡ„еҶ…е®№е°ұжҳҜиҜ·жұӮдҪ“зҡ„еҶ…е®№пјҢиҜ·жұӮдҪ“е’ҢиҜ·жұӮеӨҙд№Ӣй—ҙжҳҜжңүдёҖдёӘз©әиЎҢйҡ”ејҖгҖӮжӯӨж—¶жөҸи§ҲеҷЁеҸ‘йҖҒзҡ„жҳҜPOSTиҜ·жұӮгҖӮ

  • GETиҜ·жұӮиҜ·жұӮеҸӮж•°еңЁиҜ·жұӮиЎҢдёӯпјҢжІЎжңүиҜ·жұӮдҪ“пјҢPOSTиҜ·жұӮиҜ·жұӮеҸӮж•°еңЁиҜ·жұӮдҪ“дёӯгҖӮ
  • GETиҜ·жұӮиҜ·жұӮеҸӮж•°еӨ§е°ҸжңүйҷҗеҲ¶пјҢPOSTжІЎжңүгҖӮ

е“Қеә”ж•°жҚ®ж јејҸвҖӢ

е“Қеә”ж•°жҚ®жҖ»е…ұеҲҶдёәдёүйғЁеҲҶеҶ…е®№пјҢеҲҶеҲ«жҳҜе“Қеә”иЎҢгҖҒе“Қеә”еӨҙгҖҒе“Қеә”дҪ“гҖӮ

е“Қеә”иЎҢвҖӢ

е“Қеә”ж•°жҚ®зҡ„з¬¬дёҖиЎҢ,е“Қеә”иЎҢеҢ…еҗ«дёүеқ—еҶ…е®№пјҢеҲҶеҲ«жҳҜHTTP/1.1[HTTPеҚҸи®®еҸҠзүҲжң¬]гҖҒ200[е“Қеә”зҠ¶жҖҒз Ғ]гҖҒ ok[зҠ¶жҖҒз Ғзҡ„жҸҸиҝ°]гҖӮ

е“Қеә”еӨҙвҖӢ

з¬¬дәҢиЎҢејҖе§ӢпјҢж јејҸдёәkeyпјҡvalueеҪўејҸпјҢе“Қеә”еӨҙдёӯдјҡеҢ…еҗ«иӢҘе№ІдёӘеұһжҖ§пјҢеёёи§Ғзҡ„HTTPе“Қеә”еӨҙжңүпјҡ

Content-TypeпјҡиЎЁзӨәиҜҘе“Қеә”еҶ…е®№зҡ„зұ»еһӢпјҢдҫӢеҰӮtext/htmlпјҢimage/jpegпјӣ
Content-LengthпјҡиЎЁзӨәиҜҘе“Қеә”еҶ…е®№зҡ„й•ҝеәҰпјҲеӯ—иҠӮж•°пјүпјӣ
Content-EncodingпјҡиЎЁзӨәиҜҘе“Қеә”еҺӢзј©з®—жі•пјҢдҫӢеҰӮgzipпјӣ
Cache-ControlпјҡжҢҮзӨәе®ўжҲ·з«Ҝеә”еҰӮдҪ•зј“еӯҳпјҢдҫӢеҰӮmax-age=300иЎЁзӨәеҸҜд»ҘжңҖеӨҡзј“еӯҳ300з§’

е“Қеә”дҪ“вҖӢ

жңҖеҗҺдёҖйғЁеҲҶгҖӮеӯҳж”ҫе“Қеә”ж•°жҚ®гҖӮ

е“Қеә”зҠ¶жҖҒз ҒвҖӢ

зҠ¶жҖҒз ҒеӨ§е…Ёпјҡhttps://cloud.tencent.com/developer/chapter/13553

еҲҶзұ»иҜҙжҳҺ
