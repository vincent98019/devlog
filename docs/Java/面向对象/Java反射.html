JavaеҸҚе°„

JAVAеҸҚе°„жңәеҲ¶жҳҜеңЁиҝҗиЎҢзҠ¶жҖҒдёӯпјҢеҜ№д»»ж„ҸдёҖдёӘзұ»пјҢйғҪиғҪзҹҘйҒ“иҝҷдёӘзұ»зҡ„жүҖжңүеұһжҖ§е’Ңж–№жі•пјӣеҜ№д»»ж„ҸдёҖдёӘеҜ№иұЎпјҢйғҪиғҪеӨҹи°ғз”Ёе®ғзҡ„д»»ж„Ҹж–№жі•е’ҢеұһжҖ§пјҢиҝҷз§ҚеҠЁжҖҒиҺ·еҸ–дҝЎжҒҜд»ҘеҸҠеҠЁжҖҒи°ғз”ЁеҜ№иұЎж–№жі•зҡ„еҠҹиғҪз§°дёәjavaиҜӯиЁҖзҡ„еҸҚе°„жңәеҲ¶гҖӮ

еҸҚе°„(Reflect)жҳҜеңЁиҝҗиЎҢж—¶еҠЁжҖҒи®ҝй—®зұ»дёҺеҜ№иұЎзҡ„жҠҖжңҜпјҢжҳҜJDK1.2зүҲжң¬еҗҺзҡ„й«ҳзә§зү№жҖ§пјҢйҡ¶еұһдәҺjava.lang.reflectпјҢеӨ§еӨҡж•°JavaжЎҶжһ¶йғҪеҹәдәҺеҸҚе°„е®һзҺ°еҸӮж•°й…ҚзҪ®гҖҒеҠЁжҖҒжіЁе…Ҙзӯүзү№жҖ§гҖӮ

еҠ иҪҪе®Ңзұ»д№ӢеҗҺпјҢеңЁйҳҹдёӯе°ұдә§з”ҹдәҶдёҖдёӘClassзұ»еһӢзҡ„еҜ№иұЎ(дёҖдёӘзұ»еҸӘжңүдёҖдёӘClassеҜ№иұЎ)пјҢиҝҷдёӘеҜ№иұЎеҢ…еҗ«дәҶзҙҜзҡ„е®Ңж•ҙз»“жһ„дҝЎжҒҜгҖӮ

  • еҸҜд»ҘеңЁзЁӢеәҸиҝҗиЎҢиҝҮзЁӢдёӯпјҢж“ҚдҪңиҝҷдәӣеҜ№иұЎгҖӮ
  • еҸҜд»Ҙи§ЈиҖҰпјҢжҸҗй«ҳзЁӢеәҸзҡ„еҸҜжү©еұ•жҖ§гҖӮ

еҸҚе°„жңәеҲ¶зҡ„зӣёе…ізұ»вҖӢ

  • Classзұ»пјҡд»ЈиЎЁзұ»зҡ„е®һдҪ“пјҢеңЁиҝҗиЎҢзҡ„Javaеә”з”ЁзЁӢеәҸдёӯиЎЁзӨәзұ»е’ҢжҺҘеҸЈпјҢClassеҜ№иұЎиЎЁзӨәжҹҗдёӘзұ»еҠ иҪҪеҗҺеңЁе Ҷдёӯзҡ„еҜ№иұЎ
  • Fieldзұ»пјҡд»ЈиЎЁзұ»зҡ„жҲҗе‘ҳеҸҳйҮҸпјҲжҲҗе‘ҳеҸҳйҮҸд№ҹз§°дёәзұ»зҡ„еұһжҖ§пјү
  • Methodзұ»пјҡд»ЈиЎЁзұ»зҡ„ж–№жі•
  • Constructorзұ»пјҡд»ЈиЎЁзұ»зҡ„жһ„йҖ ж–№жі•

Fieldзұ»гҖҒMethodзұ»е’ҢConstructorзұ»дёӢйқўдёүдёӘзұ»еҜ№иұЎйғҪжңүsetAccessible() ж–№жі•пјҢдҪңз”ЁжҳҜеҗҜеҠЁе’ҢзҰҒз”Ёи®ҝй—®е®үе…ЁжЈҖжҹҘзҡ„ејҖе…ігҖӮ

  • trueиЎЁзӨәеҸҚе°„зҡ„еҜ№иұЎеңЁдҪҝз”Ёж—¶еҸ–ж¶Ҳи®ҝй—®жЈҖжҹҘпјҢжҸҗй«ҳеҸҚе°„ж•ҲзҺҮгҖӮ
  • falseиЎЁзӨәеҸҚе°„зҡ„еҜ№иұЎжү§иЎҢи®ҝй—®жЈҖжҹҘгҖӮ

еҸҚе°„зҡ„дјҳзӮ№е’ҢзјәзӮ№вҖӢ

  • дјҳзӮ№пјҡеҸҜд»ҘеҠЁжҖҒзҡ„еҲӣе»әе’ҢдҪҝз”ЁеҜ№иұЎпјҢдҪҝз”ЁзҒөжҙ»пјҢжІЎжңүеҸҚе°„жңәеҲ¶пјҢжЎҶжһ¶жҠҖжңҜе°ұеӨұеҺ»еә•еұӮж”Ҝж’‘гҖӮ
  • зјәзӮ№пјҡдҪҝз”ЁеҸҚе°„еҹәжң¬жҳҜи§ЈйҮҠжү§иЎҢпјҢеҜ№жү§иЎҢйҖҹеәҰжңүеҪұе“ҚгҖӮ

еҸҚе°„жңәеҲ¶жҳҜJavaе®һзҺ°еҠЁжҖҒиҜӯиЁҖзҡ„е…ій”®гҖӮ

  • йқҷжҖҒеҠ иҪҪпјҡзј–иҜ‘ж—¶еҠ иҪҪзӣёе…ізҡ„зұ»пјҢеҰӮжһңжІЎжңүеҲҷжҠҘй”ҷпјҢдҫқиө–жҖ§еӨӘејәгҖӮ
  • еҠЁжҖҒеҠ иҪҪпјҡиҝҗиЎҢжҳҜеҠ иҪҪйңҖиҰҒзҡ„зұ»пјҢеҰӮжһңиҝҗиЎҢж—¶дёҚз”ЁиҜҘзұ»пјҢеҚідҪҝдёҚеӯҳеңЁд№ҹдёҚжҠҘй”ҷпјҢйҷҚдҪҺдәҶдҫқиө–жҖ§гҖӮ

зұ»иў«еҠ иҪҪзҡ„ж—¶жңәвҖӢ

  1. йқҷжҖҒеҠ иҪҪпјҡеҲӣе»әеҜ№иұЎ(new)
  2. йқҷжҖҒеҠ иҪҪпјҡеҪ“еӯҗзұ»иў«еҠ иҪҪж—¶пјҢзҲ¶зұ»д№ҹдјҡеҠ иҪҪ
  3. йқҷжҖҒеҠ иҪҪпјҡи°ғз”Ёзұ»дёӯзҡ„йқҷжҖҒжҲҗе‘ҳ
  4. еҠЁжҖҒеҠ иҪҪпјҡеҸҚе°„

йқҷжҖҒеҠ иҪҪвҖӢ

public static viod main(String[] args) {
    Cat cat = new Cat();     // жӯӨж—¶пјҢеҰӮжһңжІЎжңүCatзұ»пјҢеҲҷдјҡеңЁзј–иҜ‘зҡ„ж—¶еҖҷе°ұжҠҘй”ҷ
    cat.eat();
}

еҠ иҪҪйҳ¶ж®өвҖӢ

JVMеңЁиҜҘйҳ¶ж®өдё»иҰҒжҳҜе°Ҷеӯ—иҠӮз Ғд»ҺдёҚеҗҢзҡ„ж•°жҚ®жәҗ(еҸҜиғҪжҳҜclassж–Үд»¶пјҢд№ҹеҸҜиғҪжҳҜjarеҢ…гҖҒзҪ‘з»ңзӯү)иҪ¬еҢ–дёәдәҢиҝӣеҲ¶еӯ—иҠӮжөҒеҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢе№¶з”ҹжҲҗдёҖдёӘиҜҘиЎЁиҜҘзұ»зҡ„ClassеҜ№иұЎгҖӮ

иҝһжҺҘйҳ¶ж®өвҖӢ

йӘҢиҜҒвҖӢ

зӣ®зҡ„жҳҜдёәдәҶзЎ®дҝқclassж–Үд»¶зҡ„еӯ—иҠӮжөҒдёӯеҢ…еҗ«зҡ„дҝЎжҒҜз¬ҰеҗҲеҪ“еүҚиҷҡжӢҹжңәзҡ„иҰҒжұӮпјҢе№¶дё”дёҚдјҡеҚұе®іиҷҡжӢҹжңәиҮӘиә«зҡ„е®үе…ЁгҖӮеҢ…жӢ¬дҪҶдёҚйҷҗдәҺпјҡж–Үд»¶ж јејҸйӘҢиҜҒ(жҳҜеҗҰд»Ҙйӯ”ж•°0xcafebabeејҖеӨҙ)гҖҒе…ғж•°жҚ®йӘҢиҜҒгҖҒеӯ—иҠӮз ҒйӘҢиҜҒе’Ңз¬ҰеҸ·еј•з”ЁйӘҢиҜҒгҖӮ

еҸҜд»ҘиҖғиҷ‘дҪҝз”Ё-Xverify:none еҸӮж•°жқҘе…ій—ӯеӨ§йғЁеҲҶзҡ„зұ»йӘҢиҜҒжҺӘж–ҪпјҢзј©зҹӯиҷҡжӢҹжңәзұ»еҠ иҪҪзҡ„ж—¶й—ҙгҖӮ

еҮҶеӨҮвҖӢ

JVMдјҡеңЁиҜҘйҳ¶ж®өеҜ№йқҷжҖҒеҸҳйҮҸеҲҶй…ҚеҶ…еӯҳе№¶еҲқе§ӢеҢ–(еҜ№еә”ж•°жҚ®зұ»еһӢзҡ„еҲқе§ӢеҖјпјҢеҰӮ 0гҖҒ0LгҖҒnullгҖҒfalse)гҖӮиҝҷдәӣеҸҳйҮҸжүҖдҪҝз”Ёзҡ„еҶ…еӯҳйғҪдјҡеңЁж–№жі•еҢәдёӯиҝӣиЎҢеҲҶй…ҚгҖӮ

еҰӮжһңжҳҜfinal дҝ®йҘ°зҡ„йқҷжҖҒеёёйҮҸпјҢдјҡзӣҙжҺҘиөӢжңҖз»Ҳзҡ„еҖјгҖӮ

и§ЈжһҗвҖӢ

иҷҡжӢҹжңәе°ҶеёёйҮҸжұ еҶ…зҡ„з¬ҰеҸ·еј•з”ЁжӣҝжҚўдёәзӣҙжҺҘеј•з”Ёзҡ„иҝҮзЁӢгҖӮ

жҜ”еҰӮпјҡдёҖдёӘеёёйҮҸзҡ„еј•з”ЁжҳҜAзұ»зҡ„дёҖдёӘеҜ№иұЎпјҢеңЁиҝҷдёҖжӯҘд№ӢеүҚиҝҷдёӘеёёйҮҸеј•з”Ёзҡ„жҳҜдёҖдёӘз¬ҰеҸ·жқҘжҢҮеҗ‘Aзұ»пјҢдҪҶдёҚжҳҜеҜ№иұЎпјӣиҖҢеҲ°иҝҷдёҖжӯҘд№ӢеҗҺпјҢдјҡе°ҶиҜҘеёёйҮҸеј•з”Ёе…·дҪ“зҡ„еҶ…еӯҳең°еқҖеҖјгҖӮпјҲз¬ҰеҸ·жҳҜзӣёеҜ№зҡ„пјҢеј•з”ЁжҳҜз»қеҜ№зҡ„пјү

еҲқе§ӢеҢ–вҖӢ

еҲ°еҲқе§ӢеҢ–йҳ¶ж®өпјҢжүҚзңҹжӯЈејҖе§Ӣжү§иЎҢзұ»дёӯзҡ„Javaд»Јз ҒпјҢжӯӨйҳ¶ж®өжҳҜжү§иЎҢ<clinit>() ж–№жі•зҡ„иҝҮзЁӢгҖӮиҜҘж–№жі•жҳҜз”ұзј–иҜ‘еҷЁжҢүиҜӯеҸҘеңЁжәҗж–Үд»¶дёӯеҮәзҺ°зҡ„йЎәеәҸпјҢдҫқж¬ЎиҮӘеҠЁж”¶йӣҶзұ»дёӯжүҖжңүзҡ„йқҷжҖҒеҸҳйҮҸзҡ„иөӢеҖјеҠЁдҪңе’ҢйқҷжҖҒд»Јз Ғеқ—дёӯзҡ„иҜӯеҸҘпјҢе№¶иҝӣиЎҢеҗҲе№¶гҖӮ

иҷҡжӢҹжңәдјҡдҝқиҜҒдёҖдёӘзұ»зҡ„<clinit>() ж–№жі•еңЁеӨҡзәҝзЁӢзҡ„зҺҜеўғдёӯиў«жӯЈзЎ®зҡ„еҠ й”ҒгҖҒеҗҢжӯҘпјҢеҰӮжһңеӨҡдёӘзәҝзЁӢеҗҢж—¶еҺ»еҲқе§ӢеҢ–дёҖдёӘзұ»пјҢйӮЈд№ҲдјҡеҸӘжңүдёҖдёӘзәҝзЁӢеҺ»жү§иЎҢиҝҷдёӘзұ»зҡ„<clinit>() ж–№жі•пјҢе…¶д»–зәҝзЁӢйғҪйңҖиҰҒйҳ»еЎһзӯүеҫ…пјҢзӣҙеҲ°жҙ»еҠЁзәҝзЁӢжү§иЎҢ<clinit>() ж–№жі•е®ҢжҜ•гҖӮ

Classзұ»вҖӢ

Classд№ҹжҳҜзұ»пјҢз»§жүҝдёҺObjectзұ»пјҢClassзұ»еҜ№иұЎдёҚжҳҜnewеҮәжқҘзҡ„пјҢжҳҜеңЁеҠ иҪҪзұ»зҡ„ж—¶еҖҷзі»з»ҹеҲӣе»әзҡ„пјҢеӣ жӯӨдёҖдёӘзұ»зҡ„ClassеҜ№иұЎеңЁеҶ…еӯҳдёӯеҸӘжңүдёҖд»ҪпјҢеӣ дёәзұ»еҸӘеҠ иҪҪдёҖж¬ЎгҖӮ

йҖҡиҝҮClassеҜ№иұЎеҸҜд»ҘиҺ·еҫ—иҜҘзұ»зҡ„з»“жһ„дҝЎжҒҜе’ҢдёҖдәӣе…¶д»–дҝЎжҒҜгҖӮ

Classзұ»еҜ№иұЎжҳҜж”ҫеңЁе Ҷдёӯзҡ„пјҢзұ»зҡ„еӯ—иҠӮз ҒдәҢиҝӣеҲ¶ж•°жҚ®жҳҜж”ҫеңЁж–№жі•еҢәзҡ„гҖӮ

еңЁзұ»з¬¬дёҖж¬Ўиў«еҠ иҪҪж—¶пјҢдјҡиҝӣе…ҘClassLoaderзұ»дёӯзҡ„loadClassж–№жі•еҲӣе»әClassеҜ№иұЎгҖӮ

жңүClassзұ»еҜ№иұЎзҡ„зұ»еһӢжңүпјҡ

  • еӨ–йғЁзұ»гҖҒжҲҗе‘ҳеҶ…йғЁзұ»гҖҒйқҷжҖҒеҶ…йғЁзұ»гҖҒеұҖйғЁеҶ…йғЁзұ»гҖҒеҢҝеҗҚеҶ…йғЁзұ»
  • жҺҘеҸЈ
  • ж•°з»„
  • жһҡдёҫ

иҺ·еҸ–жҲҗе‘ҳеҸҳйҮҸвҖӢ

  • getFields()пјҡиҺ·еҸ–жүҖжңүpublicдҝ®йҘ°зҡ„жҲҗе‘ҳеҸҳйҮҸпјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

  • getField(String name)пјҡиҺ·еҸ–жҢҮе®ҡеҗҚз§°зҡ„publicдҝ®йҘ°зҡ„жҲҗе‘ҳеҸҳйҮҸпјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

  • getDeclaredFields()пјҡиҺ·еҸ–жүҖжңүзҡ„жҲҗе‘ҳеҸҳйҮҸпјҢдёҚиҖғиҷ‘дҝ®йҘ°з¬ҰпјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

  • getDeclaredField(String name)пјҡиҺ·еҸ–жҢҮе®ҡеҗҚз§°зҡ„жҲҗе‘ҳеҸҳйҮҸпјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

иҺ·еҸ–жһ„йҖ ж–№жі•вҖӢ

  • getConstructor(Class...<?> parameterTypes)пјҡиҺ·еҫ—иҜҘзұ»дёӯдёҺеҸӮж•°зұ»еһӢеҢ№й…Қзҡ„е…¬жңүжһ„йҖ ж–№жі•гҖӮ

  • getConstructors()пјҡиҺ·еҫ—иҜҘзұ»зҡ„жүҖжңүе…¬жңүжһ„йҖ ж–№жі•гҖӮ

  • getDeclaredConstructor(Class...<?> parameterTypes)пјҡиҺ·еҫ—иҜҘзұ»дёӯдёҺеҸӮж•°зұ»еһӢеҢ№й…Қзҡ„жһ„йҖ ж–№жі•гҖӮ

  • getDeclaredConstructors()пјҡиҺ·еҫ—иҜҘзұ»жүҖжңүжһ„йҖ ж–№жі•гҖӮ

иҺ·еҸ–жҲҗе‘ҳж–№жі•вҖӢ

  • getMethod(String name, Class...<?> parameterTypes)пјҡиҺ·еҫ—иҜҘзұ»жҹҗдёӘе…¬жңүзҡ„ж–№жі•пјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

  • getDeclaredMethod(String name, Class...<?> parameterTypes)пјҡиҺ·еҫ—иҜҘзұ»жҹҗдёӘж–№жі•пјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

  • getDeclaredMethods()пјҡиҺ·еҫ—иҜҘзұ»жүҖжңүж–№жі•пјҢеҢ…жӢ¬зҲ¶зұ»зҡ„гҖӮ

иҺ·еҸ–зұ»гҖҒзұ»еҜ№иұЎвҖӢ

  • зұ»еҗҚ.classпјҡйҖҡиҝҮзұ»еҗҚзҡ„еұһжҖ§classиҺ·еҸ–пјҢеӨҡз”ЁдәҺеҸӮж•°зҡ„дј йҖ’гҖӮеҗҢдёҖдёӘеӯ—иҠӮз Ғж–Үд»¶(*.class)еңЁдёҖж¬ЎзЁӢеәҸиҝҗиЎҢиҝҮзЁӢдёӯпјҢеҸӘдјҡиў«еҠ иҪҪдёҖж¬ЎпјҢдёҚи®әйҖҡиҝҮе“ӘдёҖз§Қж–№ејҸиҺ·еҸ–зҡ„ClassеҜ№иұЎйғҪжҳҜеҗҢдёҖдёӘгҖӮ

  • еҜ№иұЎ.getClass()пјҡжӯӨж–№жі•еңЁObjectзұ»дёӯе®ҡд№үгҖӮеӨҡз”ЁдәҺеҜ№иұЎзҡ„иҺ·еҸ–еӯ—иҠӮз Ғзҡ„ж–№ејҸгҖӮ

  • forName(String className)пјҡж №жҚ®зұ»еҗҚиҝ”еӣһзұ»зҡ„еҜ№иұЎпјҢеӨҡз”ЁдәҺй…ҚзҪ®ж–Үд»¶пјҢе°Ҷзұ»еҗҚе®ҡд№үеңЁй…ҚзҪ®ж–Үд»¶дёӯгҖӮиҜ»еҸ–ж–Үд»¶пјҢеҠ иҪҪзұ»гҖӮ

  • getName()пјҡиҺ·еҫ—зұ»зҡ„е®Ңж•ҙи·Ҝеҫ„еҗҚеӯ—гҖӮ

  • getClassLoader()пјҡиҺ·еҫ—зұ»зҡ„еҠ иҪҪеҷЁгҖӮ

  • newInstance()пјҡеҲӣе»әзұ»зҡ„е®һдҫӢгҖӮи°ғз”Ёзҡ„жҳҜзұ»зҡ„ж— еҸӮжһ„йҖ гҖӮ

  • getPackage()пјҡиҺ·еҫ—зұ»зҡ„еҢ…гҖӮ

  • getSimpleName()пјҡиҺ·еҫ—зұ»зҡ„еҗҚеӯ—гҖӮ

  • getSuperclass()пјҡиҺ·еҫ—еҪ“еүҚзұ»з»§жүҝзҡ„зҲ¶зұ»зҡ„еҗҚеӯ—гҖӮ

  • CastпјҡжҠҠеҜ№иұЎиҪ¬жҚўжҲҗд»ЈиЎЁзұ»жҲ–жҳҜжҺҘеҸЈзҡ„еҜ№иұЎгҖӮ

иҺ·еҸ–зұ»дёӯжіЁи§ЈвҖӢ

  • getAnnotation(Class<A> annotationClass)пјҡиҝ”еӣһиҜҘзұ»дёӯдёҺеҸӮж•°зұ»еһӢеҢ№й…Қзҡ„е…¬жңүжіЁи§ЈеҜ№иұЎгҖӮ

  • getAnnotations()пјҡиҝ”еӣһиҜҘзұ»жүҖжңүзҡ„е…¬жңүжіЁи§ЈеҜ№иұЎгҖӮ

иҺ·еҸ–ClassеҜ№иұЎзҡ„ж–№ејҸвҖӢ

з¬¬дёҖз§Қпјҡе·ІзҹҘдёҖдёӘзұ»зҡ„е…Ёзұ»еҗҚпјҢдё”иҜҘзұ»еңЁзұ»и·Ҝеҫ„дёӢпјҢеҸҜд»ҘйҖҡиҝҮClassзұ»зҡ„йқҷжҖҒж–№жі•forName() ж–№жі•иҺ·еҸ–пјҢеҰӮжһңиҺ·еҸ–дёҚеҲ°пјҢдјҡжҠӣеҮәClassNotFoundException ејӮеёёгҖӮ

дёҖиҲ¬з”ЁдәҺй…ҚзҪ®ж–Үд»¶пјҢиҜ»еҸ–зұ»зҡ„е…Ёи·Ҝеҫ„пјҢеҠ иҪҪзұ»гҖӮ

з¬¬дәҢз§ҚпјҡиӢҘе·ІзҹҘе…·дҪ“зҡ„зұ»пјҢеҸҜд»ҘйҖҡиҝҮзҙҜзҡ„classиҺ·еҸ–пјҢиҜҘж–№ејҸжңҖе®үе…ЁеҸҜйқ пјҢзЁӢеәҸжҖ§иғҪжңҖй«ҳгҖӮ

з¬¬еӣӣз§ҚпјҡйҖҡиҝҮClassLoaderзҡ„ж–№ејҸгҖӮ

public class ReflectTest {
    public static void main(String[] args) throws Exception {
        // 1. иҺ·еҸ–Userзҡ„ClassеҜ№иұЎ
        Class<?> userClass = Class.forName("run.arbor.reflect.User");

        // 2. йҖҡиҝҮpublicзҡ„ж— еҸӮжһ„йҖ еҷЁеҲӣе»әе®һдҫӢ
        Object user = userClass.newInstance();
        System.out.println("publicзҡ„ж— еҸӮжһ„йҖ пјҡ" + user);

        // 3. йҖҡиҝҮpublicзҡ„жңүеҸӮжһ„йҖ еҲӣе»әе®һдҫӢ
        Constructor<?> constructor = userClass.getConstructor(String.class);
        // жӯӨж—¶зҡ„constructorеҜ№иұЎе°ұд»ЈиЎЁдәҶpublic User(String name)жһ„йҖ еҷЁ
        Object arbor = constructor.newInstance("arbor");
        System.out.println("publicзҡ„жңүеҸӮжһ„йҖ пјҡ" + arbor);

        // 4. йҖҡиҝҮйқһpublicзҡ„жңүеҸӮжһ„йҖ еҲӣе»әе®һдҫӢ
        Constructor<?> declaredConstructor = userClass.getDeclaredConstructor(String.class, int.class);
        // 4.1 еӣ дёәжҳҜз§Ғжңүзҡ„пјҢж— жі•дҪҝз”ЁиҜҘеҜ№иұЎеҲӣе»әеҜ№иұЎпјҢжҠӣеҮәдәҶjava.lang.IllegalAccessException
        // 4.2 жүҖд»ҘйңҖиҰҒдҪҝз”ЁжҡҙеҠӣеҸҚе°„жқҘи°ғз”Ёйқһpublicзҡ„ж–№жі•жҲ–иҖ…еұһжҖ§пјҢtrueдёәжҡҙеҠӣеҸҚе°„пјҢй»ҳи®Өдёәfalse
        declaredConstructor.setAccessible(true);
        Object arbor1 = declaredConstructor.newInstance("arbor", 25);
        System.out.println("йқһpublicзҡ„жңүеҸӮжһ„йҖ пјҡ" + arbor1);
    }
}

public class ReflectTest {

        Class<?> bossClass = Class.forName("run.arbor.reflect.Boss");

        // Object invoke = say.invoke(boss,  20, "arbor", 'еӨ§');
        Object invoke = say.invoke(null, 20, "arbor", 'еӨ§');

Fieldзұ» жҲҗе‘ҳеҸҳйҮҸвҖӢ

и®ҫзҪ®еҖјвҖӢ

  • void set(Object obj, Object value)пјҡе°ҶжҢҮе®ҡзҡ„еҜ№иұЎеҸӮж•°дёӯз”ұжӯӨ FieldеҜ№иұЎиЎЁзӨәзҡ„еӯ—ж®өи®ҫзҪ®дёәжҢҮе®ҡзҡ„ж–°еҖјгҖӮ

иҺ·еҸ–еҖјвҖӢ

  • void get(Object obj)пјҡиҝ”еӣһз”ұиҜҘ FieldиЎЁзӨәзҡ„еӯ—ж®өеңЁжҢҮе®ҡеҜ№иұЎдёҠзҡ„еҖјгҖӮ

еҝҪз•Ҙи®ҝй—®жқғйҷҗдҝ®йҘ°з¬Ұзҡ„е®үе…ЁжЈҖжҹҘпјҡ

  • void setAccessible(true)пјҡжҡҙеҠӣеҸҚе°„пјҢе°ҶжӯӨеҸҚе°„еҜ№иұЎзҡ„ accessibleж Үеҝ—и®ҫзҪ®дёәжҢҮзӨәзҡ„еёғе°”еҖјгҖӮ

иҺ·еҸ–зӣёе…ідҝЎжҒҜвҖӢ

  • getModifiers() пјҡд»ҘintеҪўејҸиҝ”еӣһдҝ®йҘ°з¬ҰпјҢ0 й»ҳи®Өдҝ®йҘ°з¬ҰгҖҒ1 publicгҖҒ2 privateгҖҒ4 protectedгҖҒ8 staticгҖҒ16 finalгҖӮ

еҰӮжһңжҳҜ public static еҲҷиҝ”еӣһзӣёеҠ еҫ—еҲ°зҡ„ж•°еӯ—пјҢе°ұжҳҜ 1 + 8 = 9гҖӮеҫ—еҲ°зҡ„з»“жһңдёҚдјҡйҮҚеӨҚпјҢжүҖд»ҘиғҪз®—еҮәжңүе“Әдәӣдҝ®йҘ°з¬ҰгҖӮ

  • getType()пјҡд»ҘClassеҪўејҸиҝ”еӣһзұ»еһӢгҖӮ
  • getName()пјҡиҝ”еӣһеұһжҖ§еҗҚгҖӮ
public class ReflectTest {
    public static void main(String[] args) throws Exception {
        // 1. иҺ·еҸ–Studentзҡ„ClassеҜ№иұЎ
        Class<?> studentClass = Class.forName("run.arbor.reflect.Student");

        // 2. йҖҡиҝҮpublicзҡ„ж— еҸӮжһ„йҖ еҷЁеҲӣе»әе®һдҫӢ
        Object student = studentClass.newInstance();
        System.out.println("ж— еҸӮжһ„йҖ еҲӣе»әзҡ„еҜ№иұЎпјҡ" + student);

        Field age = studentClass.getField("age");
        System.out.println("иҺ·еҸ–еҜ№иұЎзҡ„ageеұһжҖ§еҖјпјҡ" + age.get(student));
        age.set(student, 25);
        System.out.println("дҝ®ж”№ageеұһжҖ§еҗҺзҡ„еҜ№иұЎпјҡ" + student);

        // 4. дҪҝз”ЁеҸҚе°„ж“ҚдҪңnameеұһжҖ§
        Field name = studentClass.getDeclaredField("name");
        // 4.1 еӣ дёәnameеұһжҖ§жҳҜз§Ғжңүзҡ„пјҢеҰӮжһңжғіи®ҝй—®пјҢйңҖиҰҒжҡҙеҠӣеҸҚе°„
        name.setAccessible(true);
        // 4.2 еӣ дёәnameеұһжҖ§жҳҜstaticзҡ„пјҢеҰӮжһңжғіи®ҫзҪ®еұһжҖ§пјҢеҸҜд»ҘдёҚдј е…ҘеҜ№иұЎпјҢдј е…ҘдёҖдёӘnullеҚіеҸҜ
        // name.set(student, "arbor");
        name.set(null, "arbor");
        System.out.println("дҝ®ж”№nameеұһжҖ§еҗҺзҡ„еҜ№иұЎпјҡ" + student);

    }
}
