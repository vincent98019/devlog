Javaзұ»Math

java.lang.Mathзұ»еҢ…еҗ«з”ЁдәҺжү§иЎҢеҹәжң¬ж•°еӯҰиҝҗз®—зҡ„ж–№жі•пјҢеҰӮеҲқзӯүжҢҮж•°гҖҒеҜ№ж•°гҖҒе№іж–№ж №е’Ңдёүи§’еҮҪж•°гҖӮзұ»дјјиҝҷж ·зҡ„е·Ҙе…·зұ»пјҢе…¶жүҖжңүж–№жі•еқҮдёәйқҷжҖҒж–№жі•пјҢе№¶дё”дёҚдјҡеҲӣе»әеҜ№иұЎпјҢи°ғз”Ёиө·жқҘйқһеёёз®ҖеҚ•гҖӮ

еёёз”Ёж–№жі•вҖӢ

abs(...)пјҡз»қеҜ№еҖјпјҢиҝ”еӣһдј е…ҘеҖјзҡ„з»қеҜ№еҖјгҖӮ

pow(double a, double b)пјҡжұӮе№ӮпјҢиҝ”еӣһaзҡ„bж¬Ўж–№гҖӮ

ceil(double a)пјҡеҗ‘дёҠеҸ–ж•ҙпјҢиҝ”еӣһеӨ§дәҺзӯүдәҺеҸӮж•°зҡ„жңҖе°Ҹзҡ„ж•ҙж•°гҖӮ

floor(double a)пјҡеҗ‘дёӢеҸ–ж•ҙпјҢиҝ”еӣһе°ҸдәҺзӯүдәҺеҸӮж•°зҡ„жңҖеӨ§зҡ„ж•ҙж•°гҖӮ

round(...)пјҡеӣӣиҲҚдә”е…ҘпјҢиҝ”еӣһжңҖжҺҘиҝ‘еҸӮж•°зҡ„ж•ҙж•°гҖӮ(зӣёеҪ“дәҺеӣӣиҲҚдә”е…Ҙж–№жі•)гҖӮ

sqrt(double a)пјҡжұӮејҖж–№пјҢиҝ”еӣһdoubleзұ»еһӢзҡ„еӣӣиҲҚдә”е…ҘеҗҺзҡ„е№іж–№ж №гҖӮ

random()пјҡжұӮйҡҸжңәж•°пјҢиҝ”еӣһдёҖдёӘ0.0-1.0зҡ„йҡҸжңәж•°гҖӮеӨ§дәҺзӯүдәҺ0иҖҢе°ҸдәҺ1гҖӮ

max(...)пјҡжұӮдёӨдёӘж•°зҡ„жңҖеӨ§еҖјпјҢиҝ”еӣһдј е…ҘдёӨдёӘеҸӮж•°жңҖеӨ§зҡ„йӮЈдёӘгҖӮ min(...)пјҡжұӮдёӨдёӘж•°зҡ„жңҖе°ҸеҖјпјҢиҝ”еӣһдј е…ҘдёӨдёӘеҸӮж•°жңҖе°Ҹзҡ„йӮЈдёӘгҖӮ

ж–№жі•жЎҲдҫӢвҖӢ

жұӮйҡҸжңәж•°вҖӢ

йңҖжұӮпјҡ дҪҝз”Ёrandom()ж–№жі•иҺ·еҸ–дёҖдёӘ2-7д№Ӣй—ҙзҡ„дёҖдёӘйҡҸжңәж•ҙж•°пјҢеҢ…еҗ«2е’Ң7гҖӮ

в‘  жӯЈеёёжқҘиҜҙпјҢеҰӮжһңжҲ‘д»¬жғіиҰҒ0-9д№Ӣй—ҙзҡ„дёҖдёӘйҡҸжңәж•ҙж•°пјҢдјҡжҠҠrandomзҡ„иҝ”еӣһеҖјзӣҙжҺҘд№ҳд»Ҙ10пјҢеҶҚејәиҪ¬дёәintзұ»еһӢгҖӮ в‘Ў йӮЈеҰӮжһңжғіиҰҒ1-10пјҢеҸӘйңҖиҰҒе°Ҷв‘ зҡ„з»“жһңеҠ 1еҚіеҸҜпјҢзұ»жҺЁеҸҜд»ҘеҸ‘зҺ°жҲ‘д»¬зҡ„йңҖжұӮеҸҜд»ҘжӢҶеҲҶжҲҗпјҡиҺ·еҸ–дёҖдёӘ0-5д№Ӣй—ҙзҡ„ж•ҙж•°пјҢз„¶еҗҺеҶҚеҠ 2пјҢеҫ—еҲ°зҡ„е°ұжҳҜ2-7д№Ӣй—ҙзҡ„ж•ҙж•°гҖӮ в‘ў йӮЈеҰӮдҪ•иҺ·еҸ–0-5д№Ӣй—ҙзҡ„ж•ҙж•°е‘ўпјҹиҝҷйҮҢжҳҜеҢ…еҗ«5зҡ„пјҢдҪҶжҳҜrandomж–№жі•зҡ„з»“жһңжҳҜ0 <= x < 1зҡ„е°Ҹж•°пјҢеҢ…еҗ«0пјҢдёҚеҢ…еҗ«1гҖӮжүҖд»Ҙе®һйҷ…дёҠжҲ‘д»¬йңҖиҰҒиҺ·еҸ–зҡ„жҳҜ0 <= x < 6зҡ„ж•ҙж•°гҖӮ в‘Ј 0 <= x < 1зҡ„е°Ҹж•°пјҢд№ҳд»Ҙ10пјҢеҫ—еҲ°зҡ„е°ұжҳҜ0 <= x < 10зҡ„еҖјпјӣд№ҳд»Ҙ9пјҢ0 <= x < 9зҡ„еҖјпјӣд»ҘжӯӨзұ»жҺЁпјҢиҝҷйҮҢеҸҜд»ҘжӢҝrandomж–№жі•зҡ„иҝ”еӣһеҖјд№ҳд»Ҙ6пјҢеҫ—еҲ°зҡ„е°ұжҳҜ0 <= x < 6зҡ„еҖјгҖӮ в‘Ө е°ҶиҜҘеҖјеҠ 2пјҢе№¶ејәиҪ¬дёәintзұ»еһӢзҡ„ж•°еҖјпјҢеҚіеҸҜгҖӮ в‘Ҙ з”ұжӯӨеҸҜд»Ҙеҫ—еҲ°дёҖдёӘе…¬ејҸпјҡ (int) (Math.random() * (b - a + 1) + a) в‘Ұ жҠҠ2еёҰе…ҘaпјҢ7еёҰе…ҘbпјҢеҸҜд»Ҙеҫ—еҲ°пјҡ (int) (Math.random() * (7 - 2 + 1) + 2) (int) (Math.random() * 6 + 2)

public static void main(String[] args) {
	int a = 2;
	int b = 7;
	int res = (int) (Math.random() * (b - a + 1) + a);
	System.out.println("2-7д№Ӣй—ҙзҡ„йҡҸжңәж•°дёәпјҡ" + res);
}
